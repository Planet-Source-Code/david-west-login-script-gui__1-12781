1. There is a component that comes with this program.  It is ASP User by Persits Software.  You must register this componet before it can be used.

Copy the component to C:\Windows\System
then run regsvr32 c:\Windows\System\ASPUser.dll

2. You must create a Policies Directory in each of the Domain Controllers Netlogon Directories...

3. If you want to use the Outlook Profile Creation Utility w/in the Program you need to copy newprof.exe to all the domain controllers netlogon directories...
